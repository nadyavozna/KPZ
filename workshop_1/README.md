# Звіт по практичній роботі №1

## Мета роботи
Ознайомитися з базовими командами та принципами роботи з системою контролю версій Git шляхом проходження інтерактивного навчального курсу. Сформувати практичні навички використання Git у консольному середовищі для виконання основних операцій.

## Завдання
Пройти інтерактивний курс «Git How To» за посиланням: https://githowto.com/uk.

### Мінімальні вимоги (обов’язкова частина):
Повністю пройти Частину 1 курсу, яка охоплює базові операції:
- Ініціалізація репозиторію;
- Додавання та коміти файлів;
- Перегляд історії;
- Скасування змін;
- Робота з гілками.

## Виконання
Частина 1: Основи роботи з Git
Я пройшла першу частину курсу, яка включала основні операції роботи з Git. У процесі виконання я виконала наступні кроки:
### Ініціалізація репозиторію
Створила новий Git-репозиторій, використовуючи команду git init.
### Додавання та коміти файлів
Додала файли до репозиторію за допомогою команди git add . та зробила перший коміт через git commit -m "Initial commit".
### Перегляд історії
Переглянула історію комітів за допомогою команди git log.
### Скасування змін
Використала команди для скасування змін:
git restore для скасування незаписаних змін;
git reset для скасування проіндексованих змін.
### Робота з гілками
Створила нову гілку за допомогою git branch.
